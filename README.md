# 📊 Probabilidade e Estatística - Resumos e Anotações

> Anotações e resumos de Probabilidade e Estatística. O repositório abrange desde Estatística Descritiva até Regressão e Correlação, construindo a base matemática essencial para tomadas de decisão e estudos avançados em Machine Learning e Engenharia de Software.

## 🎯 Objetivos de Estudo

Este repositório documenta a jornada de aprendizado para:
* Compreender todas as fases do método estatístico (Problema, Planejamento, Obtenção dos Dados, Organização, Resumo, Análise e Tomada de Decisões).
* Entender a importância da amostragem e dos conceitos fundamentais de probabilidade.
* Desenvolver habilidades práticas em Análise Exploratória de Dados e Inferência Estatística.
* Utilizar a Ciência Estatística como ferramenta lógica e estruturada para tomadas de decisões.
